# Week 3 — Exercises

Three exercises, ~7 hours total. Do them in order; each builds on the previous.

| # | Exercise | Time |
|---|----------|-----:|
| 1 | [Three views, three ways](./exercise-01-three-views-three-ways.md) — same screen as FBV, `View`, `ListView` | 2h |
| 2 | [`ModelForm` with validation](./exercise-02-modelform-with-validation.md) — `ArticleForm` with `clean_<field>` + `clean()` | 2.5h |
| 3 | [Login and protected views](./exercise-03-login-and-protected-views.md) — real login + an owner-only edit screen | 2.5h |

Work in the same Django project you used for Week 2's mini-project (`crunchwriter` v0). If you didn't finish W2, do W2's mini-project first — these exercises assume the `Article`, `Category`, and admin are already in place.
