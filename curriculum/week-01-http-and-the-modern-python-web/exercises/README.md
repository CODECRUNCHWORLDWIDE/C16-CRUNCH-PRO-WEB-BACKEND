# Week 1 — Exercises

Short, focused drills. Each one should take 20–40 minutes. Do them in order; later ones assume earlier ones.

## Index

1. **[Exercise 1 — Raw HTTP with `nc`](exercise-01-raw-http-with-netcat.md)** — type an HTTP request character by character into netcat and inspect the response. (~25 min)
2. **[Exercise 2 — A stdlib HTTP server](exercise-02-stdlib-server.py)** — write a 30-line server using only `http.server`. (~30 min)
3. **[Exercise 3 — WSGI hello world](exercise-03-wsgi-hello-world.py)** — a real WSGI app served by `wsgiref`. (~25 min)

## How to work the exercises

- Read the prompt. Skim, don't memorize.
- **Type the code yourself**. Do not copy-paste. Muscle memory is the entire point.
- Run it. See the output. Read the error if it crashed.
- If you get stuck for more than 10 minutes, peek at the inline hints.
- Only check the official solution after you've made it run.

There are no solutions checked in. The course is open source — solutions live in forks and Gists from other learners. Search GitHub for `c16-week-01 site:github.com` after you've finished to compare.
