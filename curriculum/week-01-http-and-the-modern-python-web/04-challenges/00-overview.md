# Week 1 — Challenges

The exercises drill basics. **Challenges stretch you.** Each takes 60–120 minutes and produces something you can commit to your portfolio.

## Index

1. **[Challenge 1 — Status Code Explorer](./challenge-01-status-code-explorer.md)** — build a tool that deliberately triggers every HTTP status code family from a single test harness. (~90 min)
2. **[Challenge 2 — Hand-Written HTTP Response](./challenge-02-write-the-response.md)** — type out a valid HTTP/1.1 response by hand and verify it round-trips through `curl`. (~75 min)

Challenges are optional. If you skip them, you can still pass the week. If you do them, you'll be measurably ahead of someone who didn't.
