# Week 5 — Challenges

One stretch challenge this week, ~3 hours. Optional but recommended. The discipline of reading SQL critically and reproducing it in the ORM is the single most useful skill for the rest of Phase 2 — and for any future job where the previous engineer dropped to raw SQL for a report that should have been a manager method.

| # | Challenge | Time |
|---|-----------|-----:|
| 1 | [Recreate a complex SQL as ORM](./challenge-01-recreate-a-complex-sql-as-orm.md) — translate a hand-written analytics SQL into a single ORM queryset | ~3h |

Challenges are graded on the **closeness of the emitted SQL to the original** and on the **defensibility of the differences**. A queryset that runs and produces correct numbers but emits SQL with two extra subselects is fine — name the differences in the write-up.
